# 2048 Lite - 轻量版经典复刻

## 游戏简介

《2048 Lite》是**经典 2048 游戏的复刻版**，我们从最基础的玩法出发，简化了一切不必要的元素，去除了广告和复杂资源，只留下你最喜欢的数字合并挑战。**没有任何吃内存的东西**，也没有任何多余的内容，纯粹的游戏体验，随时可以打开玩一局。
**2048 不只关于数字合并，而是在有限空间里对抗随机性。**

## 主要特点

### 1. **经典复刻**

* 没有加入花里胡哨的功能和复杂的 UI，**复刻最经典的玩法**：通过合并相同数字的方块，目标是合成 2048。
* 目标很简单，就是让你享受**最纯粹的2048**。
* 方块颜色递进参考了**经典 2048 配色**：低阶数字保持原版米色到橙色过渡，高阶数字切入金黄色，再向深灰色收束，既保留熟悉观感，也兼容更大的数字继续增长。

### 2. **极致轻量**

* 游戏**只有一个 HTML 文件**，没有额外的图片、音频或任何重型框架。**没有广告**，不占用内存，不拖慢速度。
* 完全依靠 **HTML、CSS 和 JavaScript** 纯手工打造，极小的体积，打开瞬间加载完毕。**不需要安装**，直接在浏览器中运行。**没有图片素材**，即便数字变得很大，依然能完美显示（极大数采用指数形式）。
* 采用干净简洁的设计，**没有多余的装饰**。游戏背景温和，数字方块颜色变化直观，合并时有流畅的动画。

### 3. **跨平台支持**

* **PC端**：WASD 或方向键来控制，适合桌面浏览器。
* **移动端**：支持滑动操作

### 4. **简单的记录系统**

* **当前分数**：实时记录你的分数。
* **历史记录**：主页显示最高分、用时和记录时间，随时能回顾过去的成绩。
* **无打扰**：没有提示或弹窗，专注在游戏本身。



## 为什么这样做？

* **没必要复杂化**：2048 是一个已经验证过的经典游戏，完全不需要**加任何额外的功能**或“特色”。
* **极致简化**：没有无关的动画或场景切换，没有花哨的背景音乐和复杂的 UI，只有最原始的数字方块合并和挑战。
* **没有广告**：市面上很多类似的游戏充斥着广告，而《2048 Lite》不做任何广告插入，只为了给玩家提供一个**干净、专注**的游戏环境。


《2048 Lite》是**经典的 2048 游戏复刻版**，没有复杂的界面或多余的功能，只为提供一个**最纯粹的游戏体验**。它是为喜欢经典游戏的玩家准备的，**没有广告，简单、纯粹、轻量**。无论是在桌面还是移动设备上，你都可以随时打开它，享受最简单的数字合并挑战。
